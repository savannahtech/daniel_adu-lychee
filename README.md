The backend is using nextjs server-side to handle video generation:

## Getting Started

First, run the development server:

```bash
yarn install
yarn dev

or

npm run install
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


# With Docker
