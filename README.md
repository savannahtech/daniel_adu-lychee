## Getting Started

First, run the development server:

```bash
yarn install
yarn dev

or

npm run install
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


# With Single Environment Deployment

```shell
docker compose build
docker compose up
```
